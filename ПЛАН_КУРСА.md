```python
def safe_float_input(prompt: str) -> float:
    while True:
        try:
            value = float(input(prompt))
            if value <= 0:
                raise ValueError("Должно быть больше нуля")
            return value
        except ValueError as e:
            print(f"Ошибка: {e}. Попробуй снова.")
```

### Урок 1.4 — ООП: Классы (5-7 дней)
**Изучи:**
- `class`, `__init__`, `self`
- Методы экземпляра, `@classmethod`, `@staticmethod`
- `__str__`, `__repr__`
- Наследование
- `@property`
- `@dataclass`

**Задание — класс Trade:**
```python
from dataclasses import dataclass
from datetime import datetime

@dataclass
class Trade:
    coin: str
    buy_price: float
    sell_price: float
    amount: float
    created_at: str = ""

    def __post_init__(self):
        self.created_at = datetime.now().isoformat()

    @property
    def profit(self) -> float:
        return (self.sell_price - self.buy_price) * self.amount

    @property
    def profit_percent(self) -> float:
        return ((self.sell_price - self.buy_price) / self.buy_price) * 100

    def to_dict(self) -> dict: ...

    @classmethod
    def from_dict(cls, data: dict) -> "Trade": ...
```

**Задание — класс TradeJournal:**
```python
class TradeJournal:

    def get_best_trade(self) -> Trade: ...
    def get_worst_trade(self) -> Trade: ...
    def get_stats(self) -> dict: ...  # total_trades, total_profit, win_rate

    def load(self) -> None: ...
```

### Урок 1.5 — Тестирование с pytest (3 дня)
**Изучи:**
- `pip install pytest`
- Файл `test_*.py`, функции `test_*`
- `assert`
- Фикстуры: `@pytest.fixture`
- Запуск: `pytest -v`

**Задание — минимум 10 тестов:**
```python
def test_profit_positive():
    trade = Trade("BTC", buy_price=50000, sell_price=55000, amount=1)
    assert trade.profit == 5000

def test_profit_negative():
    trade = Trade("ETH", buy_price=3000, sell_price=2500, amount=2)
    assert trade.profit == -1000

def test_profit_percent():
    trade = Trade("BTC", buy_price=100, sell_price=110, amount=1)
    assert trade.profit_percent == pytest.approx(10.0)

def test_journal_win_rate():
    journal = TradeJournal()
    journal.add_trade(Trade("BTC", 100, 110, 1))
    journal.add_trade(Trade("ETH", 100, 90, 1))
    assert journal.get_stats()["win_rate"] == 50.0
```
